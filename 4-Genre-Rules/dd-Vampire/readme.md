## Example: Sabbat Vampire Creation – [SAMPLE ONLY]

2. Faction: Black Hand (Org Approval Required)

4. Morality: Path of Metamorphosis

6. Abilities: 55 total

13. Negative Traits: Arrogant, Bestial  
14. Flaws: Dark Secret (2), Enemy (3), Addiction (2)  
15. Derangement: Megalomania  
16. Optional: Lowered Morality for +2 Freebies  
17. Base Freebie Points: 5  
18. Starting XP: 30  
19. Blood: 13 Traits, Spend 2 per round  
20. Willpower: 6
